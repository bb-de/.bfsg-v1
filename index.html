Barrierefreiheit-Widget

Dies ist das optimierte Barrierefreiheit-Widget für die Webflow-Integration. Der blaue Accessibility-Button in der unteren rechten Ecke öffnet das Widget-Panel.

Funktionen des Widgets:

  • Kontrastanpassungen: Dunkler Kontrast, Heller Kontrast, Hohe Sättigung, Monochrom
  • Textanpassungen: Verschiedene Schriftgrößen, Zeilenabstand und Buchstabenabstand
  • Lesehilfen: Leseführung, großer Cursor, Lesemaske
  • Voreingestellte Profile: Für Sehbehinderung, kognitive Behinderung und ADHS

Testanleitung:

  1. Klicken Sie auf den blauen Accessibility-Button unten rechts
  2. Wählen Sie eine der Anpassungsoptionen oder ein Profil
  3. Beobachten Sie die Änderungen auf der Seite
  4. Sie können die Einstellungen zurücksetzen mit dem Button "Zurücksetzen"

Integration in Webflow:

Um das Widget in Ihre Webflow-Website einzubinden, kopieren Sie den folgenden Code und fügen Sie ihn am Ende des Body-Tags ein:

<script src="https://github.com/bb-de/.bfsg-v1/accessibility-widget-final.js"></script>

Ersetzen Sie 'https://ihre-domain.com/' mit der tatsächlichen URL, unter der Sie das Widget hosten.
